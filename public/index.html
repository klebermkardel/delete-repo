Gerar Token do GitHub

Para utilizar este sistema, você precisa de um token de acesso do GitHub. Siga os passos abaixo para gerar o token:

  1. Acesse o site do GitHub e faça login na sua conta.
  2. Clique na sua foto de perfil no canto superior direito e selecione "Settings" (Configurações).
  3. No menu lateral esquerdo, selecione "Developer settings" (Configurações de desenvolvedor).
  4. Clique em "Personal access tokens" (Tokens de acesso pessoal).
  5. Na página de tokens de acesso pessoal, clique em "Generate new token" (Gerar novo token).
  6. Forneça uma descrição para o token, selecione as permissões necessárias e clique em "Generate token" (Gerar token).
  7. Copie o token gerado e cole-o no campo acima.
